 4.4 FormView／DetailsViewコントロール
------------------------------------------------------------
・FormView...... 自由形式の単票ビュー
・DetailsView... テンプレート固定の単票ビュー
※一般的には、より自由度の高いFormViewコントロールを利用することの方が多いため、本書でもFormViewコントロールのみを解説。

■FormViewコントロールによる単票ビューの作成手順■
1．ページレイアウトをjデザインする
  ・デザインビューからFormViewコントロールを配置。
  ・SqlDataSourceを設定。

2．FormViewコントロールの挙動と見栄えを設定する
  ・タスクメニューから、[ページングを有効にする]の項目にチェックを追加。
  ・[オートフォーマット...]メニューから「プロフェッショナル」フォーマットを適用。
    ※GridViewとは異なり、FOｒｍＶｉｅｗコントロールでは該当するテンプレートが存在するか、テンプレート内に該当するコマンドボタンがあるかどうかによって、新規登録や更新、削除の有無／無効を決定する。
      タスクメニューからこれら機能の有効／無効を設定することはできない。

3．FormFiewコントロールのテンプレートを設定する
  [FormViewコントロールで利用可能なテンプレート]
  ・ItemTemplate......... 個別アイテムの表示（標準モード）★
  ・EditItemTemplate..... 個別アイテムの更新（編集モード）★
  ・InsertItemTemplate... 個別アイテムの新規登録（新規登録モード）★
  ・HeaderTemplate....... ヘッダー部分★
  ・FooterTemplate....... フッター部分
  ・PagerTemplate........ ページャー部分
  ・EmptyDataTemplate.... データが存在しない部分
  ※本節では★の付いたテンプレートを定義する。

  ・バインド式の入力

  ・CommandName／CommandArgumentプロパティ
    	New					： [新規]ボタン
    	Insert				： [挿入]ボタン
    	Delete				： [削除]ボタン
    	Edit				： [編集]ボタン
    	Update				： [更新]ボタン
    	Cancel				： [キャンセル]ボタン
    -----------------------------------------------
    	Page	／	Next	： 次ページへ
        		／	Prev	： 前ページへ
        		／	First	： 先頭ページへ
        		／	Last	： 最終ページへ

  ・FormViewコントロールで単票フォームで行われた動作をアプリケーションから制御するためのイベントを公開。
    [FormViewコントロールで利用可能な主なイベント]
    ※※※ 一旦省略 ※※※

  ・更新処理後に画面を遷移する
  =============================================================================================
    protected void FormView1_ItemUpdated( object sender, FormViewUpdatedEventArgs e )
    {
        Response.Redirect("GridViewBook.aspx");
    }
  =============================================================================================

  ・更新処理後に特定の表示モードに遷移する
  =============================================================================================
    protected void FormView1_ItemUpdated( object sender, FormViewUpdatedEventArgs e )
    {
        fv.DefaultMode = FormViewMode.Insert;	// 新規フォームに移動
    }
  =============================================================================================
   [DefaultModeプロパティの設定値（FormViewMode列挙体のメンバー）]
    ・Edit....... 編集モード
    ・Insert..... 挿入モード
    ・ReadOnly... 表示モード（デフォルト）

  ・ItemXxxxxingイベントハンドラーを利用することで、パラメーター値をアプリケーションから動的にセット
  =============================================================================================
    protected void fv_ItemUpdating( object sender, FormViewUpdateEventArgs e )
    {
        // 単価欄に負数が入力された場合
        if( Int32.Parse( e.NewValue["price"].ToString() ) <= 0 )
        {
            e.NewValues["price"] = e.OldValues["price"];
        }

        // [別解]
        var txtPrice = (TextBox) fv.FindControl("priceTextBox");
        if( Int32.Parse( txtPrice.Text ) <= 0 )
        {
            e.NewValues["price"] = e.OldValues["price"];
        }
    }
  =============================================================================================
